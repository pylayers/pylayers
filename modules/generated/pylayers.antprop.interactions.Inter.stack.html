pylayers.antprop.interactions.Inter.stack¶

Inter.stack(data=array([], dtype=float64), idx=0, isdata=True)[source]¶

stack data and the associated idx

Parameters:

data : np.array()

data to stack

idx :

index to stack

isdata: bool

False if you just want to stack idx (only used for intE class )

Examples

>>> from pylayers.antprop.rays import *
>>> import numpy as np
>>> I=Inter()
>>> data = np.array(([3,4,5]))
>>> idx = 0
>>> I.stack(data,idx)
>>> I.data
array([3, 4, 5])
>>> I.idx
[0]
>>> data = np.array(([3,4,5],[7,8,9]))
>>> idx = [1,2]
>>> I.stack(data,idx)
>>> I.data
array([[3, 4, 5],
       [3, 4, 5],
       [7, 8, 9]])
>>> I.idx
[0, 1, 2]
